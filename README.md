# Simple Calculator

A simple web-based calculator built using HTML, CSS, and JavaScript.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero
- Easy-to-use interface

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download or clone this repository.
2. Open `index.html` in your web browser.

## Future Improvements

- Input validation
- Keyboard support
- Dark mode
- Responsive design
